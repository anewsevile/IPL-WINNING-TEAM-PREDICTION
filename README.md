# IPL Team Prediction 🏏

This project simulates IPL matches and predicts the winning team using player-level data.

## Features
- Data cleaning and preprocessing
- Feature engineering (bowling type, age, encoding)
- Player strength calculation
- Team strength aggregation
- Match and tournament simulation
- Visualization using matplotlib

## Approach
Since real performance metrics were not available, a simulation-based approach was used:
- Player strengths were calculated using weighted scores
- Teams were formed and evaluated based on total strength
- Matches were simulated probabilistically
- A full IPL season was simulated to determine the winner

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib

## Output
- Team strength comparison
- Points table
- Predicted IPL winner
